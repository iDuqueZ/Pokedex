# PokeDex

Pokemon Wiki

## Built With
-  [React](https://reactjs.org/)
-  [Parcel](https://parceljs.org/)
- [PokeApi](https://pokeapi.co/)

## Getting Started

- Fork this repo, clone to your desktop, and install dependencies:
  ```sh
  npm install
  ```
- Start the server, and navigate to `localhost:1234`
  ```sh
  npm run start
  ```
